# CompleteDiscordQuest
این اسکریپت برای کامل کردن خودکار Questهای جدید دیسکورد استفاده می‌شود. بعد از قبول Quest در Discord، کد داخل کنسول اجرا می‌شود و با شبیه‌سازی فعالیت‌های لازم مثل تماشای ویدیو یا بازی کردن، پیشرفت Quest را تکمیل می‌کند تا کاربر بتواند پاداش آن را دریافت کند. استفاده از آن فقط در Discord Desktop امکان‌پذیر است.
